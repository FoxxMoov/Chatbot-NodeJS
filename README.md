# Hello World Server

===============

## Setup

```
$ git clone https://gitlab.eemi.tech/corentin.ravet/chatbot-node.git
$ npm install
$ cd chatbot-node
$ npm run start
```

# What it is, what it does

Server that responds "Hello World" or something else to HTTP GET and POST request
